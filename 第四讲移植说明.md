## 📝 调试记录（2026-04-19）

### 1. 语音模块启动优化

**问题**：系统初始化时语音播报"门已上锁"异常

**原因**：语音模块上电后需要一定启动时间，且第一次通信用于唤醒串口

**解决方案**：
```c
void Audio_Init(void)
{
    USART3_Init();
    delay_ms(1200);       // 等待模块启动
    Audio_WakeUp();       // 发送唤醒命令
}
```

### 2. 字模格式问题修复

**问题**：部分汉字显示异常（"等待"、"欢迎回家"等）

**原因**：字模取模方向与LCD扫描方向不匹配

**解决方案**：
- 统一字模格式为**低位在前**
- 区分汉字和ASCII字模的扫描方向：
  - 汉字：`0x01 << j`（低位在前）
  - ASCII：`0x80 >> j`（高位在前）

### 3. 密码输入错误处理修复

**问题**：输入错误管理员密码时未清空输入

**修复**：在错误处理中增加 `Key_Clear()`
```c
case 1:  // 修改密码模式
    if(String_Check(key_temp, password_cmd, 6))
    {
        // 验证成功
    }
    else
    {
        Audio_Play(9);      // "管理员密码错误"
        Key_Clear();        // 清空输入
    }
```

### 4. 自动锁门语音添加

**问题**：5秒自动锁门后没有语音提示

**修复**：在 `Lock_Proc()` 中检测状态变化并播放语音
```c
void Lock_Proc(void)
{
    if(lock_flag == 1 && lock_flag_old == 0)
    {
        Audio_Play(2);  // "门已上锁"
    }
    lock_flag_old = lock_flag;
    Lock(lock_flag);
}
```

### 5. 倒计时时间修改

**修改**：密码错误锁定时间从 **15秒** 改为 **30秒**

```c
// main.c
u16 time30s = 30;  // 原为 time15s = 15

// timer.c
if(--time30s == 0)  // 30秒倒计时
{
    time30s = 30;
    password_error = 0;
}
```
