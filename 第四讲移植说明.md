# 第四讲移植说明（语音+RFID功能）

## 📋 新增功能

第四讲在第三讲基础上增加了**语音播报**和**RFID刷卡**功能：

- ✅ **SYN6288语音模块** - 播报语音提示（门已开启、门已上锁等）
- ✅ **RC522 RFID模块** - 刷卡开锁功能
- ✅ **矩阵键盘扩展** - 支持管理员密码、修改密码、录入卡片
- ✅ **中文显示** - LCD显示中文提示信息
- ✅ **安全机制** - 密码错误3次锁定30秒

## 🔌 硬件连接

### 语音模块（USART3）
| 功能 | 引脚 | 说明 |
|------|------|------|
| USART3_TX | PB10 | 连接语音模块RX |
| USART3_RX | PB11 | 连接语音模块TX |

### RFID模块（SPI2）
| 功能 | 引脚 | 说明 |
|------|------|------|
| SCK | PB13 | SPI2时钟 |
| MISO | PB14 | SPI2数据输入 |
| MOSI | PB15 | SPI2数据输出 |
| SDA | PB12 | 片选信号 |
| RST | PB11 | 复位信号 |
| VCC | 3.3V | 供电 |
| GND | GND | 共地 |

### 矩阵键盘扩展功能
```
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ A │  ← R1
├───┼───┼───┼───┤
│ 4 │ 5 │ 6 │ B │  ← R2
├───┼───┼───┼───┤
│ 7 │ 8 │ 9 │ C │  ← R3
├───┼───┼───┼───┤
│ * │ 0 │ # │ D │  ← R4
└───┴───┴───┴───┘
  ↑   ↑   ↑   ↑
 C1  C2  C3  C4
```

- **A键**：进入管理员模式
- **B键**：录入RFID卡片
- **C键**：清除输入
- **D键**：确认/OK
- **#键**：修改密码模式
- **\*键**：返回/退出

## 🚀 使用方法

### 硬件连接汇总
```
【LCD显示屏】
SCK  → PC10
MOSI → PC12
RES  → PA1
DC   → PA3
CS   → PA4
BL   → PB9

【舵机】
信号线 → PA0
电源   → 5V外置
地线   → GND

【矩阵键盘】
R1 → PE13, R2 → PE15, R3 → PD9, R4 → PD11
C1 → PE11, C2 → PE9,  C3 → PE7, C4 → PC5

【语音模块】
RX → PB10 (USART3_TX)
TX → PB11 (USART3_RX)
VCC → 5V
GND → GND

【RFID模块】
SCK  → PB13
MISO → PB14
MOSI → PB15
SDA  → PB12
RST  → PB11
VCC  → 3.3V
GND  → GND
```

### 操作流程
1. **开锁方式**：
   - 输入6位密码（默认：123456），按D键确认
   - 或刷已录入的RFID卡片

2. **管理员模式**（按A键）：
   - 输入管理员密码（默认：666888）
   - B键：录入新卡片
   - #键：修改开锁密码

3. **修改密码**（管理员模式下按#键）：
   - 输入新6位密码
   - 按D键确认

## ⚠️ 注意事项

1. **语音模块供电**：需要5V供电，电流不小于200mA
2. **RFID模块供电**：3.3V供电，不要接5V
3. **卡片存储**：最多存储10张卡片
4. **密码存储**：存储在Flash中，掉电保存
5. **USART冲突**：USART3用于语音，USART2预留

## 📝 调试记录（2026-04-19）

### 1. 语音模块启动优化

**问题**：系统初始化时语音播报"门已上锁"异常

**原因**：语音模块上电后需要一定启动时间，且第一次通信用于唤醒串口

**解决方案**：
```c
void Audio_Init(void)
{
    USART3_Init();
    delay_ms(1200);       // 等待模块启动
    Audio_WakeUp();       // 发送唤醒命令
}
```

### 2. 字模格式问题修复

**问题**：部分汉字显示异常（"等待"、"欢迎回家"等）

**原因**：字模取模方向与LCD扫描方向不匹配

**解决方案**：
- 统一字模格式为**低位在前**
- 区分汉字和ASCII字模的扫描方向：
  - 汉字：`0x01 << j`（低位在前）
  - ASCII：`0x80 >> j`（高位在前）

### 3. 密码输入错误处理修复

**问题**：输入错误管理员密码时未清空输入

**修复**：在错误处理中增加 `Key_Clear()`
```c
case 1:  // 修改密码模式
    if(String_Check(key_temp, password_cmd, 6))
    {
        // 验证成功
    }
    else
    {
        Audio_Play(9);      // "管理员密码错误"
        Key_Clear();        // 清空输入
    }
```

### 4. 自动锁门语音添加

**问题**：5秒自动锁门后没有语音提示

**修复**：在 `Lock_Proc()` 中检测状态变化并播放语音
```c
void Lock_Proc(void)
{
    if(lock_flag == 1 && lock_flag_old == 0)
    {
        Audio_Play(2);  // "门已上锁"
    }
    lock_flag_old = lock_flag;
    Lock(lock_flag);
}
```

### 5. 倒计时时间修改

**修改**：密码错误锁定时间从 **15秒** 改为 **30秒**

```c
// main.c
u16 time30s = 30;  // 原为 time15s = 15

// timer.c
if(--time30s == 0)  // 30秒倒计时
{
    time30s = 30;
    password_error = 0;
}
```
