# 第三讲移植说明（密码锁功能）

## 📋 新增功能

第三讲在第二讲基础上增加了**密码锁功能**：

- ✅ **4×4矩阵键盘** - 输入数字密码（0-9）
- ✅ **密码验证** - 6位密码，正确开锁，错误保持上锁
- ✅ **LCD显示** - 显示密码输入界面，*号 masking
- ✅ **任务调度器** - TIM3定时器实现多任务调度

## 🔑 矩阵键盘引脚配置

### 行输出（推挽输出）
| 行 | CH32V30x | STM32F407VET6 |
|----|----------|---------------|
| R1 | PE13 | **PE13** ✅ |
| R2 | PE15 | **PE15** ✅ |
| R3 | PD9 | **PD9** ✅ |
| R4 | PD11 | **PD11** ✅ |

### 列输入（上拉输入）
| 列 | CH32V30x | STM32F407VET6 |
|----|----------|---------------|
| C1 | PE11 | **PE11** ✅ |
| C2 | PE9 | **PE9** ✅ |
| C3 | PE7 | **PE7** ✅ |
| C4 | PC5 | **PC5** ✅ |

### 键盘布局
```
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │   │  ← R1
├───┼───┼───┼───┤
│ 4 │ 5 │ 6 │   │  ← R2
├───┼───┼───┼───┤
│ 7 │ 8 │ 9 │   │  ← R3
├───┼───┼───┼───┤
│ C │ 0 │ OK│   │  ← R4
└───┴───┴───┴───┘
  ↑   ↑   ↑   ↑
 C1  C2  C3  C4
```

- **数字键**：1-9, 0（输入密码）
- **C键（13）**：清除输入
- **OK键（16）**：确认密码

## 🔧 移植修改

### 1. 矩阵键盘驱动（KEY）

**文件**：`HARDWARE/KEY/key.c`

**修改内容**：
- GPIO时钟：RCC_APB2 → RCC_AHB1
- GPIO模式：GPIO_Mode_Out_PP → GPIO_Mode_OUT + GPIO_OType_PP
- GPIO速度：50MHz → 100MHz
- 添加延时消抖

### 2. 定时器驱动（TIMER）

**文件**：`HARDWARE/TIMER/timer.c`

**功能**：
- TIM2：舵机PWM控制（与第二讲相同）
- TIM3：系统滴答计数（任务调度）

**修改内容**：
- TIM2预分频：96-1 → 84-1（适应84MHz时钟）
- TIM3中断配置：添加NVIC配置

### 3. 主程序（main.c）

**功能流程**：
```
1. 系统初始化（NVIC、延时、PWM、LCD）
2. 初始上锁状态
3. 显示启动动画（进度条）
4. 显示logo和主页
5. 显示密码输入界面
6. 启动任务调度器
   - 10ms任务：按键扫描
   - 100ms任务：屏幕刷新
```

## 🎯 密码验证逻辑

```c
// 正确密码
u8 password[6] = {1, 2, 3, 4, 5, 6};

// 输入密码存储
u8 key_temp[7] = {0};

// 验证流程
1. 用户通过键盘输入6位数字
2. 每输入一位，LCD显示一个*
3. 点击OK键（16）进行验证
4. 密码正确 → Lock(0) 开锁（舵机转到1500us位置）
5. 密码错误 → Lock(1) 保持上锁（舵机保持500us位置）
```

## 🚀 使用方法

### 硬件连接

**矩阵键盘连接**：
```
键盘     STM32F407
R1     → PE13
R2     → PE15
R3     → PD9
R4     → PD11
C1     → PE11
C2     → PE9
C3     → PE7
C4     → PC5
```

**舵机连接**（与第二讲相同）：
```
舵机信号线 → PA0
舵机电源线 → 5V外部电源
舵机地线   → GND（共地）
```

**LCD连接**（与第二讲相同）：
```
SCK  → PC10
MOSI → PC12
RES  → PA1
DC   → PA3
CS   → PA4
BL   → PB9
```

### 操作流程

1. **上电**：显示"Starting"进度条，然后显示"LOCKED"
2. **输入密码**：按数字键输入6位密码（默认：123456）
3. **确认**：按OK键验证
4. **结果**：
   - 正确 → 舵机开锁
   - 错误 → 保持上锁
5. **清除**：按C键清除输入

## ⚠️ 注意事项

1. **矩阵键盘消抖**：添加了1ms延时消抖
2. **密码长度限制**：最多输入6位，超出自动截断
3. **舵机供电**：仍需5V外部电源
4. **LCD显示**：使用英文显示（字库限制）
