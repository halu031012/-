# 舵机角度方向说明

## 📐 舵机角度标准定义

### 标准舵机角度范围
```
        90°（中间位置）
         ↑
         |
0° ←-----+----→ 180°
         |
         ↓
       （270°）
```

---

## 🎯 如何确定舵机的角度方向

### 方法1：观察摇臂标记（推荐）

#### 步骤1：在舵机摇臂上贴标记
```
在舵机的摇臂（白色十字/单臂）上贴一个小胶带：
     ↓（标记）
    ──┼──
     │
```

#### 步骤2：运行程序观察
```
程序会循环显示：
0° → 90° → 180° → 90° → 0° → ...

LCD屏幕会显示当前角度：
┌────────────────┐
│  Servo Test    │
│   舵机测试     │
│  Angle:   0    │
│   角度:  0     │
└────────────────┘
```

#### 步骤3：记录位置
```
观察标记的位置变化：

位置1（0度）：标记在某方向（如左侧）
位置2（90度）：标记在中间
位置3（180度）：标记在另一方向（如右侧）

运动轨迹：
0° ──→ 90° ──→ 180° ──→ 90° ──→ 0°
左       中       右       中       左
```

---

### 方法2：使用指南针方向定义

#### 标准定义（从舵机正面看）
```
        北（0°/360°）
             ↑
             |
西（270°） ←─+─→ 东（90°）
             |
             ↓
        南（180°）
```

#### 舵机对应关系
```
0度   = 西方（左侧）
90度  = 东方（右侧）或北方（上方）
180度 = 东方（右侧）或南方（下方）
```

**注意**：不同品牌的舵机可能有不同的定义！

## 📊 常见舵机品牌的角度定义

### SG90（常见蓝色小舵机）
```
0度   = 摇臂指向左侧（逆时针极限）
90度  = 摇臂指向中间
180度 = 摇臂指向右侧（顺时针极限）
```

### MG996R（金属齿轮舵机）
```
0度   = 摇臂指向左侧（逆时针极限）
90度  = 摇臂指向中间
180度 = 摇臂指向右侧（顺时针极限）
```

### 部分数字舵机
```
可能反向：
0度   = 摇臂指向右侧
180度 = 摇臂指向左侧
```

## ⚙️ 如果方向与预期相反怎么办？

### 解决方法1：软件反转（推荐）
修改`pwm.c`中的`Servo_SetAngle`函数：

```c
void Servo_SetAngle(u8 angle)
{
    u16 pulse;
    if(angle > 180) angle = 180;

    // 方法1：直接反转角度
    angle = 180 - angle;  // 添加这一行即可反转方向

    // 角度转换为脉宽：0度=500us, 180度=2500us
    pulse = PWM_MIN + (PWM_MAX - PWM_MIN) * angle / 180;
    TIM_SetCompare1(TIM2, pulse);
}
```

### 解决方法2：硬件反转
直接将舵机摇臂拆下，反向安装即可。

## 🔍 调试技巧

### 如果舵机转动角度不足
可能需要调整PWM脉宽范围：

修改`pwm.h`：
```c
#define PWM_MIN 500   // 尝试改为 600 或 1000
#define PWM_MAX 2500  // 尝试改为 2400 或 2000
```

### 如果舵机抖动
可能PWM信号不稳定，检查：
1. 电源是否充足（至少5V 500mA）
2. 地线是否共地
3. PWM频率是否为50Hz
