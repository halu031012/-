# STM32F407 智能门锁项目

基于STM32F407VET6的智能门锁系统，从CH32V30x平台移植而来，集成了密码输入、RFID刷卡、语音播报等多种开锁方式。

## 功能特性

- ✅ **LCD显示** - ST7735屏幕显示（128x128），支持中文显示
- ✅ **PWM舵机控制** - 舵机角度控制，实现门锁开关
- ✅ **4×4矩阵键盘** - 输入数字密码（0-9）及功能键
- ✅ **密码验证** - 6位密码开锁，支持修改密码
- ✅ **RFID刷卡** - RC522模块支持刷卡开锁
- ✅ **语音播报** - SYN6288语音模块播报提示音
- ✅ **管理员模式** - 管理员密码验证后可录入卡片、修改密码
- ✅ **安全机制** - 密码错误3次锁定30秒
- ✅ **自动上锁** - 开锁后5秒自动上锁
- ✅ **中文显示** - LCD显示中文提示信息

## 硬件平台

- **MCU**: STM32F407VET6
- **主频**: 168MHz
- **LCD**: ST7735 (128x128)
- **舵机**: 标准PWM舵机
- **语音模块**: SYN6288（USART3通信）
- **RFID模块**: RC522（SPI2通信）
- **键盘**: 4×4矩阵键盘

#### 键盘布局
```
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ A │  ← R1
├───┼───┼───┼───┤
│ 4 │ 5 │ 6 │ B │  ← R2
├───┼───┼───┼───┤
│ 7 │ 8 │ 9 │ C │  ← R3
├───┼───┼───┼───┤
│ * │ 0 │ # │ D │  ← R4
└───┴───┴───┴───┘
  ↑   ↑   ↑   ↑

```

- **数字键**：1-9, 0（输入密码）
- **A键**：进入管理员模式
- **B键**：录入RFID卡片
- **C键**：清除输入
- **D键**：确认/OK
- **#键**：修改密码模式
- **\*键**：返回/退出

### 操作流程

#### 1. 上电启动
- 上电后语音播报"欢迎使用智能门锁系统"
- LCD显示"正在启动"与进度条动画
- 动画结束后进入主界面

#### 2. 开锁方式
**方式一：密码开锁**
- 输入6位密码（默认：123456）
- 按D键确认
- 密码正确：舵机开锁，语音播报"门已开启"
- 密码错误：保持上锁，语音播报"密码错误"

**方式二：RFID刷卡**
- 将已录入的RFID卡片靠近读卡器
- 验证通过后自动开锁

#### 3. 管理员模式
- 按A键进入管理员模式
- 输入管理员密码（默认：666888），按D键确认
- 验证成功后：
  - 按B键：进入录入卡片模式
  - 按#键：进入修改密码模式

#### 4. 录入卡片（管理员模式 → B键）
- 语音播报"请刷入卡片"
- 将新卡片靠近读卡器
- 卡片录入成功后有语音提示

#### 5. 修改密码（管理员模式 → #键）
- 输入新的6位密码
- 按D键确认保存

#### 6. 其他操作
- 按C键：清除当前输入
- 按*键：返回上级菜单

## 移植说明

本项目从沁恒CH32V30x平台移植到STM32F407VET6，主要修改：

1. **GPIO配置** - STM32F407使用AHB1总线时钟
2. **定时器配置** - TIM2时钟频率调整为84MHz（APB1=42MHz，定时器2倍频）
3. **SPI引脚** - SPI3引脚重新映射到PC10/PC11/PC12
4. **LCD控制引脚** - 避免与USART冲突，重新映射到PA1/PA3/PA4/PB9

## 注意事项

⚠️ **舵机供电**: 舵机需要5V外部电源，电流需求200-500mA

⚠️ **语音模块供电**: 需要5V供电，电流不小于200mA

⚠️ **RFID模块供电**: 3.3V供电，不要接5V

⚠️ **共地**: 外部电源地线必须与STM32共地

⚠️ **LCD背光**: PB9引脚控制LCD背光

⚠️ **USART冲突**: USART3用于语音，USART2预留用于其他功能

## 更新历史

### 2026-04-19 调试更新

#### 1. 语音模块启动优化
- 问题：系统初始化时语音播报异常
- 解决：增加1200ms启动延时和唤醒命令

#### 2. 字模格式问题修复
- 问题：部分汉字显示异常（"等待"、"欢迎回家"等）
- 解决：统一字模格式，区分汉字（低位在前）和ASCII（高位在前）扫描方向

#### 3. 密码输入错误处理修复
- 问题：输入错误管理员密码时未清空输入
- 解决：在错误处理中增加 `Key_Clear()`

#### 4. 自动锁门语音添加
- 问题：5秒自动锁门后没有语音提示
- 解决：在 `Lock_Proc()` 中检测状态变化并播放语音

#### 5. 倒计时时间修改
- 修改：密码错误锁定时间从15秒改为30秒

#### 6. RFID模块注释
- 为单独测试语音功能，临时注释RFID相关代码
- 恢复方法：取消对应代码块的注释即可
