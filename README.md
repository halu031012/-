# STM32F407 智能门锁项目

基于STM32F407VET6的智能门锁系统，从CH32V30x平台移植而来。

## 功能特性

- ✅ **LCD显示** - ST7735屏幕显示（128x128）
- ✅ **PWM舵机控制** - 舵机角度控制（0-180度）
- ✅ **中文显示** - 支持汉字显示和图片显示
- ✅ **进度动画** - 启动动画效果
- ✅ **矩阵键盘输入** - 4x4键盘扫描与数字录入
- ✅ **密码门锁逻辑** - 密码277516，A清空，D确认校验

## 硬件平台

- **MCU**: STM32F407VET6
- **主频**: 168MHz
- **LCD**: ST7735 (128x128)
- **舵机**: 标准PWM舵机

## 引脚配置

### LCD ST7735 (SPI3)
| 功能 | 引脚 |
|------|------|
| SPI_SCK | PC10 |
| SPI_MOSI | PC12 |
| LCD_CS | PA4 |
| LCD_DC | PA3 |
| LCD_RES | PA1 |
| LCD_BL | PB9 |

### 舵机 PWM (TIM2_CH1)
| 功能 | 引脚 |
|------|------|
| PWM信号 | PA0 |

## 硬件连接

### LCD连接
```
LCD      STM32F407
VCC  →   3.3V
GND  →   GND
SCK  →   PC10
MOSI →   PC12
RES  →   PA1
DC   →   PA3
CS   →   PA4
BL   →   PB9
```

### 舵机连接
```
Servo    STM32F407
信号线 →  PA0
电源线 →  5V（外部电源）
地线   →  GND（共地）
```

## 移植说明

本项目从沁恒CH32V30x平台移植到STM32F407VET6，主要修改：

1. **GPIO配置** - STM32F407使用AHB1总线时钟
2. **定时器配置** - 时钟频率从96MHz调整为84MHz
3. **SPI引脚** - SPI3引脚重新映射
4. **LCD控制引脚** - 避免与USART冲突

详细说明请查看 [移植说明.md](移植说明.md)

## 注意事项

⚠️ **舵机供电**: 舵机需要5V外部电源，电流需求200-500mA

⚠️ **共地**: 舵机地线必须与STM32共地

⚠️ **LCD背光**: PB9引脚控制LCD背光

## 上板验证清单

### 1) 上电与初始化
- [ ] 上电后显示“正在启动”与进度条动画
- [ ] 动画结束后进入“门已上锁，请输入密码”界面
- [ ] 输入框背景正常显示（深蓝底）

### 2) 键盘输入显示
- [ ] 每按一个数字键，输入区增加一个“*”
- [ ] 最多只接收6位输入，不会越界或显示异常

### 3) 清空功能（A键）
- [ ] 输入任意数字后按A键，输入区立即清空
- [ ] 清空后可重新输入6位密码

### 4) 错误密码校验（D键）
- [ ] 输入错误6位密码后按D键，维持“门已上锁”
- [ ] 舵机保持上锁位置（0度附近）

### 5) 正确密码校验（D键）
- [ ] 输入`277516`后按D键，显示“门已解锁”
- [ ] 舵机转到解锁位置（90度附近）

### 6) 重复测试
- [ ] 连续进行“错误→正确→错误”验证，系统稳定无死机
- [ ] 快速按键场景下无明显漏键/重触发（10ms扫描）
